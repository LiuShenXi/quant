# 版本迭代架构包

本目录保存量化平台在基线架构之后的版本化增量架构包。

已有基线:

- `架构包/`: v1.0 原始架构方案。
- `codex_修改后架构/`: v1.1-codex 落地修订版，重点是 A股 ETF 日线可信回测、Paper 前置基础设施和更保守的 M3/M4 门禁。

当前迭代:

- `v1.2-research-engine/`: 通用研究引擎架构包。目标是在不破坏 v1.1 安全边界的前提下，补齐跨市场、多频、7x24 日历、报价币种账户、bps 成本、组合级风控和通用研究报告能力。

约束:

- 架构包不是交易许可。
- 任何 paper/live/QMT/券商/交易所接入仍受根目录 `AGENTS.md`、M3b 门禁和人工授权约束。
- 版本迭代优先扩展通用框架能力，不为单一策略硬编码平台逻辑。
