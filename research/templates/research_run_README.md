## How To Start A Static Run

1. Create `research/runs/YYYY-MM-DDTHHMM__short_topic/`.
2. Copy the opening template pack into that folder.
3. Fill `run.yaml` first, then update each Markdown file.
4. Keep missing evidence explicit. Missing evidence means research-only, not
   approval to trade.

Recommended copy list:

- `00_brief.md`
- `01_thesis.md`
- `02_data_requirements.md`
- `03_experiment_plan.md`
- `07_decision_record.md`
- `run.yaml`

Use the review templates when the run reaches those gates:

- `data_audit.md`
- `backtest_validation.md`
- `risk_review.md`
- `paper_observation.md`
- `cio_decision_package.md`
